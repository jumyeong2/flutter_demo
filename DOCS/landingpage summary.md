## 2. 사전신청 모달 (EmailSignupModal)

### 제목
- **"베타 출시 알림 신청"**

### 사회적 증명
- **"이미 127명의 창업 팀이 사전 신청했습니다"**
- 파란색 배경 박스
- 사람 아이콘 2개

### 사전 신청 혜택 (3가지)
1. **출시 즉시 우선 안내**
2. **사전 신청자 전용 30% 할인 쿠폰**
3. **Pro 질문 일부 무료 체험**
   - 이전: "베타 우선 접근 또는 무료 체험 연장"
   - 변경 이유: 구체적이고 명확한 혜택 제시

### 서비스 소개 문구
- **"✓ CoSync는 말 꺼내기 어려운 질문을, 싸움이 아닌 합의로 만듭니다."**
- 특수기호: ✓ (체크마크)
- 감정적 공감: "말 꺼내기 어려운 질문"
- 가치 제안: "싸움이 아닌 합의"
- 반응형: 모바일 2줄, 데스크탑 1줄
- 왼쪽 정렬

### 입력 필드
- 이메일 주소
- 개인정보 처리방침 동의 체크박스 (필수)

### CTA 버튼
- **"알림 + 30% 쿠폰 받기"**
- 닫기 버튼: 전체 너비 (double.infinity)

### Firebase 저장 데이터
**컬렉션:** `leads`
**필드:**
- `email` (lowercase)
- `createdAt` (Timestamp)
- `source` ("meta_ad" or "landing")
- `consentPrivacy` (true)
- `utmSource`, `utmCampaign`, `utmAdset`, `utmAd` (있으면 저장)

### 이벤트 트래킹
1. `lead_modal_open` - 모달 열림
2. `lead_submit_attempt` - 제출 버튼 클릭
3. `lead_submit_success` - 저장 성공
4. `lead_submit_duplicate` - 중복 이메일
5. `lead_submit_error` - 저장 실패

### 중복 체크
- 동일 이메일 재신청 시: "이미 신청 완료되었습니다. 출시 시 동일 이메일로 안내드립니다."

## 3. 도입 효과 섹션 (Description3)

### Before (기존 방식)
1. **막연한 불안감만 존재**
2. **말로만 하는 약속**
3. **나중에 정리하자는 미루기**

### After (CoSync 도입 후)
1. **10개+ 핵심 질문 진단**
   - 정량적 표현
2. **Pro 질문 일부 무료 체험**
   - 이전: "질문 세트 확장 가능"
   - 변경 이유: 구체적 혜택 명시
3. **문서로 남는 합의**

### 핵심 포인트
- Before/After 명확한 대비
- 정량적 가치 제시 (10개+)
- 무료 체험 강조

## 4. 프로세스 섹션 (Description5)

### 메인 헤드라인
- **"감정 싸움 없이 합의하는 4단계 프로세스"**

### 서브 헤드라인
- **"CoSync는 '중간 다리' 역할을 통해 객관적인 합의를 이끌어냅니다."**

### 4단계 프로세스

#### 1. 진단 (Sync)
- **"단순한 설문이 아닙니다. 예민하지만 꼭 필요한 질문에 각자 답변합니다."**
- 아이콘: 채팅 버블
- 강조: 예민한 질문 다루기

#### 2. 리스크 시각화
- **"생각이 일치하는 부분과 조율이 필요한 부분을 데이터로 명확히 보여줍니다."**
- 아이콘: 차트
- 강조: 데이터 시각화

#### 3. 시장 표준 제안
- **"정답이 아닌 '참고 프레임'을 제공합니다. 산업 관행과 데이터를 토대로 합리적 선택을 돕습니다."**
- 아이콘: 인사이트
- 강조: 참고 프레임 제공
- 수정: "비교해" → "토대로"

#### 4. Agreement
- **"합의된 내용을 바탕으로 법적 효력을 고려한 공동창업자 합의안을 생성합니다."**
- 아이콘: 문서
- 강조: 법적 효력 고려
- 용어 변경: "Rulebook" → "Agreement", "룰북" → "합의안"

### 반응형 레이아웃
- 소형 모바일 (≤480px): 1열 세로 배치
- 태블릿 (≤1024px): 2x2 배치
- 데스크탑 (>1024px): 4개 가로 배치

### UI 개선
- 카드 좌우 여백 추가 (40px)
- 동적 높이 조정 (minHeight 사용)
- 호버 효과: 상승 애니메이션 + 그림자 강화
